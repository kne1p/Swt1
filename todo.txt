Hinweise für ÜB3
-Swing Layouts
  Aussehen unwichtig, also am einfachsten Flow Layout und JPanels zum Gruppieren verwenden
-man kann auf jedes Swing Element malen, der geforderte Bildausschnitt lässt sich also auch zB auf ein JLabel malen
-der Bildausschnitt kann beliebig gewählt werden

-zwischen der GUI Aufgabe ist auch eine Anpassung des Floyd Steinberg Algorithmus für 3*k bit pro Kanal versteckt

-Abgabe als zip das die Paketstruktur enthält: /swt1/ub3/Main.java

Tipp für Folien:
Eclipse: vorkommen von Variablen, methoden anzeigen: References>Working Set ( Ctrl+Shift+G), oder einfach markieren und dann an der Seitenleiste
Problem: OptionsEdit verwendet null-Layout, d.h. alle Elemente müssen Pixelgenau positioniert werden!
OptionsEdit: setBounds(x,y,width,height) (0,0) ist links oben, einfach "einreihen", abstände beachten
